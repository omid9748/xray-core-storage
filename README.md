
## 🛠 راهنمای استفاده قدم به قدم (فقط در ۳ دقیقه)

### 1️⃣ مرحله اول: آماده‌سازی هسته Xray روی سرور

ابتدا با دسترسی `root` وارد ترمینال سرور اوبونتو خود شوید و پوشه کاری را بسازید:

```bash
mkdir -p /opt/xray

```

سپس آخرین نسخه فایل `Xray-linux-64.zip` را دانلود کرده، نام آن را دقیقاً به **`xray.zip`** تغییر دهید و با برنامه‌های مدیریت فایل (مانند SFTP، MobaXterm، FileZilla یا پنل وب آروان) داخل مسیر زیر آپلود کنید:

`/opt/xray/`

---

### 2️⃣ مرحله دوم: دریافت و اجرای اسکریپت

دستورات زیر را به ترتیب بزنید تا به پوشه مربوطه رفته و اسکریپت راه‌انداز از گیت‌هاب دانلود شود:

```bash
cd /opt/xray

```

```bash
wget -q "[https://raw.githubusercontent.com/omid9748/xray-core-storage/main/install.sh](https://raw.githubusercontent.com/omid9748/xray-core-storage/main/install.sh)" -O install.sh

```

حالا با دستور زیر به اسکریپت مجوز اجرا داده و آن را استارت کنید:

```bash
chmod +x install.sh && sudo ./install.sh

```

---

### 3️⃣ مرحله سوم: چسباندن کانفیگ JSON

در اواسط اجرای اسکریپت، پیام دریافت کانفیگ ظاهر می‌شود. مراحل زیر را دقیق انجام دهید:

1. کانفیگ مشتری (Client JSON) خود را کپی کنید.
2. در ترمینال راست‌کلیک کنید تا کل متون پیست شود.
3. یک بار دکمه **Enter** را بزنید تا به خط خالی بعدی برود.
4. کلیدهای **`Ctrl + D`** را روی کیبورد فشار دهید تا تغییرات ذخیره و سرویس استارت شود.
