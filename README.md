# 🚀 اسکریپت راه‌اندازی پروکسی سیستمی اوبونتو (مخصوص سرورهای ایران)

این اسکریپت برای حل مشکل تحریم‌ها و فیلترینگ روی سرورهای داخل ایران (مثل ابر آروان، ایران سرور و...) طراحی شده است. با اجرای این ابزار، ترافیک کل سیستم و به خصوص مدیریت پکیج اوبونتو (`apt update` / `apt install`) از داخل هسته Xray رد شده و دسترسی به مخازن داکر، پایتون و گیت‌هاب آزاد می‌شود.

## ✨ ویژگی‌ها
- **۱۰۰٪ آفلاین در استخراج:** بدون نیاز به نصب پکیج‌های آنلاین مثل `unzip` (استفاده از پایتون داخلی سیستم).
- **اتصال سیستمی (System-wide):** ست کردن خودکار متغیرهای محیطی سیستم و پروکسی اختصاصی APT.
- **امن و سبک:** استفاده مستقیم از هسته رسمی Xray بدون ابزارهای جانبی سنگین.
