# 🚀 اسکریپت راه‌اندازی پروکسی سیستمی اوبونتو (مخصوص سرورهای ایران)

این اسکریپت برای حل مشکل تحریم‌ها و فیلترینگ روی سرورهای داخل ایران (مثل ابر آروان، ایران سرور و...) طراحی شده است. با اجرای این ابزار، ترافیک کل سیستم و به خصوص مدیریت پکیج اوبونتو (`apt update` / `apt install`) از داخل هسته Xray رد شده و دسترسی به مخازن داکر، پایتون و گیت‌هاب آزاد می‌شود.

## ✨ ویژگی‌ها
- **۱۰۰٪ آفلاین در استخراج:** بدون نیاز به نصب پکیج‌های آنلاین مثل `unzip` (استفاده از پایتون داخلی سیستم).
- **اتصال سیستمی (System-wide):** ست کردن خودکار متغیرهای محیطی سیستم و پروکسی اختصاصی APT.
- **امن و سبک:** استفاده مستقیم از هسته رسمی Xray بدون ابزارهای جانبی سنگین.

---

## 🛠 راهنمای استفاده قدم به قدم

### مرحله ۱: آماده‌سازی هسته Xray
ابتدا وارد ترمینال سرور اوبونتو خود شوید و پوشه کاری را بسازید:
```bash
mkdir -p /opt/xray


مرحله ۲: دریافت و اجرای اسکریپت
دستور زیر را بزنید تا اسکریپت راه‌انداز دانلود شود:
Bash
cd /opt/xray
wget -q "[https://raw.githubusercontent.com/omid9748/xray-core-storage/main/install.sh](https://raw.githubusercontent.com/omid9748/xray-core-storage/main/install.sh)" -O install.sh
حالا به اسکریپت مجوز اجرا داده و آن را استارت کنید:
Bash
chmod +x install.sh
sudo ./install.sh
مرحله ۳: وارد کردن کانفیگ JSON
در اواسط اجرای اسکریپت، از شما خواسته می‌شود کانفیگ خود را وارد کنید.
کانفیگ مشتری (Client JSON) خود را کپی کنید.
در ترمینال راست‌کلیک کنید تا پیست شود.
یک بار دکمه Enter را بزنید.
کلیدهای Ctrl + D را روی کیبورد فشار دهید تا ذخیره شود.
⚠️ نکته بسیار مهم برای کانفیگ: حتماً مطمئن شوید که در بخش inbounds کانفیگ شما، یک ورودی از نوع http روی پورت 20809 تنظیم شده باشد تا اوبونتو بتواند به آن متصل شود.
مرحله ۴: ریستارت و پایان
پس از اتمام موفقیت‌آمیز اسکریپت، سرور را ریستارت کنید:
Bash
sudo reboot
پس از بالا آمدن سرور، دستور sudo apt update را بزنید و از اینترنت بدون تحریم لذت ببرید!
